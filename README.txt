ZEBRA CIRCUS BLASTER — how to play
===================================

A first-person "blaster" game (runs in your web browser).
Controls: WASD / arrow keys to move, mouse to look, click to shoot, E to pick up devices.

TO PLAY
-------
1. Unzip this folder.
2. Double-click "index.html" — it opens in your browser.

If the circus tent / devices don't appear (some browsers block loading 3D
files directly from a folder), start a tiny local server instead:

  • Open Terminal (Mac) or Command Prompt (Windows) in this folder and run:

        python3 -m http.server 8000

  • Then open your browser to:  http://localhost:8000

  (On Windows, if "python3" isn't found, try "python -m http.server 8000".)

That's it — enjoy!
